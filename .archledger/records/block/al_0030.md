## Black Box: Config Layer

Loads, validates, and provides structured access to WikiMason configuration.

### Responsibilities

- Discover local config files (`wikimason.toml`, `.wikimason.toml`) by walking parent directories.
- Merge profile defaults from `profiles.py` with user overrides.
- Expose `WikiMasonConfig` frozen dataclass with `PathConfig`, `LinkConfig`, `ProfileConfig` sub-objects.
- Support named env configs stored in `~/.config/wikimason/<name>.toml`.

### Key Types

- `WikiMasonConfig`: Top-level config with root, profile, paths, links, profile_config.
- `PathConfig`: Path mapping for raw, sources, files, wiki, schema, templates, agents.
- `LinkConfig`: Link style, template, target format, label format.
- `ProfileConfig`: Tool-specific settings like frontmatter style, dot-directory creation, page layout.
