## Black Box: Source Lifecycle

Manages raw source files through scan, delta detection, verification, and lint.

### Responsibilities

- **Scan** (`source_scan.py`): Discover `Raw/Sources/*.md` files, extract frontmatter, compute hashes, build records, write manifest.
- **Delta** (`source_delta.py`): Compare current state against stored manifest, report new/changed/deleted/coverage-gap sources.
- **Verify** (`source_verify.py`): Validate manifest integrity, rehash files, lint source frontmatter.
- **Manifest** (`source_manifest.py`): Load/write `Schema/source-manifest.jsonl`.
- **Metadata** (`source_metadata.py`): Sidecar `*.wm.yaml` files, MIME detection, SHA-256 hashing, wikimason metadata embedding.

### Key Concepts

- Each source file gets a stable `source_id` and is tracked in the JSONL manifest.
- `source_coverage_report` maps sources to compiled notes for traceability.
- `source_delta` returns actionable counts per category (new, content_changed, metadata_changed, missing_coverage).
