## Black Box: CLI Layer

Responsible for parsing user commands and routing to domain logic.

### Responsibilities

- Thin `main()` entrypoint in `cli.py` that hoists global options and delegates to Typer.
- `cli_app.py` composes the Typer app and registers 20 modular command groups from `cli_groups/`.
- Global callback initializes `CliState` on the Typer context.
- JSON output mode via `--format json` for machine consumption.

### Implementation

- `cli.py`: Entrypoint, global option hoisting, error formatting.
- `cli_app.py`: App builder, command group registration.
- `cli_groups/`: 20 files, one per command domain (agents, catalog, config, daily, file, folder, index, ingest, links, note, page, property, review, root, skill, source, tag, task, template, text, vault).
- `cli_helpers.py`: Shared CLI utility functions.
- `cli_output.py`: JSON result payload formatting.
- `cli_state.py`: Runtime state object carried on Typer context.
