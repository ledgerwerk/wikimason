## Black Box: Filesystem Operations

Reads, writes, and resolves paths within the vault filesystem.

### Responsibilities

- File CRUD: `write_file`, `read_file`, `append_file`, `prepend_file`, `move_file`, `rename_file`, `delete_file`.
- File listing and search: `list_files`, `list_folders`, `search_files`.
- Path resolution: `resolve_path_in_vault`, `rel_to_vault`, `slugify_title`, path normalization.
- Note scaffolding: `new_note` creates notes with templates, frontmatter, and source/related links.
- Template rendering: `NoteScaffold` dataclass captures rendered note content.

### Key Types

- `NoteScaffold`: Frozen dataclass with path, content, kind, title, status, sources, related.
