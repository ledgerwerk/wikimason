## Black Box: Search and Indexing

Provides fuzzy search across commands, pages, sources, files, headings, tags, and aliases.

### Responsibilities

- Ranked fuzzy search using `rapidfuzz` with configurable cutoff.
- Multiple search kinds: command, page, source, file, heading, tag, alias, body.
- `SearchCandidate` dataclass with key, kind, label, path, aliases, fields.
- `SourceBackend` for searching Raw/Sources, `CommandBackend` for CLI commands.
- `rank_candidates()` applies fuzzy matching and returns scored results.
