## Black Box: Build Pipeline

Compiles derived artifacts from source notes and schema.

### Responsibilities

- `build_vault()`: Orchestrates full build: sync source counts, write catalog, rebuild indexes, render schema docs, compile AGENTS.md.
- `rebuild_indexes()`: Generate `Wiki/index.md` and per-kind section indexes from catalog entries.
- `sync_source_count()`: Ensure `source_count` frontmatter field matches actual sources list length.
- Catalog generation (`catalog.py`): Scan all compiled notes, extract metadata, write `Wiki/catalog.jsonl`.
- AGENTS.md compilation (`agents.py`): Assemble policy, workflow, purpose, L1 policy, template catalog, command reference, and profile reference. Preserve manual blocks between markers.

### Generated Artifacts

- `Wiki/catalog.jsonl`
- `Wiki/index.md`, `Wiki/Topics/index.md`, `Wiki/Concepts/index.md`, etc.
- `Schema/frontmatter-schema.md`, `Schema/command-reference.md`
- `AGENTS.md`
