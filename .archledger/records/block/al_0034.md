## Black Box: Lint and Validation

Validates compiled notes against schema contracts and checks link integrity.

### Responsibilities

- `lint_paths()` / `lint_vault()`: Run all lint rules against compiled markdown files.
- `lint_rules.py`: Per-page validation of required frontmatter fields, status values, section presence, source_count parity.
- `lint_links.py`: Validate that wikilinks in frontmatter and body resolve to existing files.
- `lint_credentials.py`: Scan for accidentally committed credentials or secrets.
- Results returned as `LintFinding` frozen dataclasses with path, line, code, message, suggestion.

### Key Types

- `LintFinding`: Immutable lint result with path, line, code, message, optional suggestion.
