## Context Interface

Human users interact with WikiMason exclusively through the command-line interface built on Typer and Click.

## Details

- Entrypoint: `wikimason.cli:main` registered as console script.
- Global options: `--config`, `--env`, `--vault`.
- All commands support `--format json` for machine-readable output.
- Fuzzy command suggestions provided via `FuzzyCommandGroup`.
