## Requirement

WikiMason must provide a CLI toolkit that operates entirely on a local filesystem to build and maintain an LLM wiki from raw source files.

## Rationale

- Coding agents need structured, source-backed knowledge accessible via filesystem reads.
- A filesystem-backed approach ensures portability, version control compatibility, and offline operation.
- The CLI surface must be deterministic and machine-consumable via JSON output.
